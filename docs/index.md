# 强智教务系统API文档

## 前言
欢迎访问强智教务系统API文档，这是一个通过对强智教务系统官方提供的智校园APP抓包得出的API文档。

此API支持查询课程表、成绩、考试信息、空教室等绝大部分会被高频使用到的信息，相比通过抓取页面获取信息效率更高，但是无法实现如抢课之类的高级功能。

开发交流QQ群：743138676

由于条件所限，部分如学籍预警、考试信息接口的API参数或返回值尚未明晰，欢迎Fork本项目进行补充。

## 查看
